LE PALAIS DE LA BEAUTÉ - MODERN FINAL WEBSITE

HOW TO OPEN:
1. Open this folder in VS Code.
2. Open index.html.
3. Right-click index.html and choose "Open with Live Server".
   If you do not have Live Server:
   - In VS Code, click Extensions
   - Search "Live Server"
   - Install the one by Ritwick Dey
   - Then right-click index.html again and open with Live Server.

FILES:
- index.html = website structure and text
- styles.css = design, colors, layout, mobile responsiveness
- script.js = animations, mobile menu, sliders
- assets = images

HOW TO CHANGE IMAGES:
Put the new image inside the assets folder and rename it exactly like the file you want to replace.

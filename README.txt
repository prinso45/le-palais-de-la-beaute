LE PALAIS DE LA BEAUTÉ - WEBSITE PROTOTYPE

SIMPLE PREVIEW:
1. Unzip this folder.
2. Open the folder.
3. Double-click index.html.
4. It opens in Google Chrome.

FILES:
- index.html
- styles.css
- script.js
- assets/

TO CHANGE IMAGES:
Put your new images inside the assets folder using these names:
- logo-beige.png
- logo-black-gold.png
- icon-logo.png
- hero-main.jpg
- salon-wide.jpg
- hair-result-1.jpg
- hair-result-2.jpg
- hair-result-3.jpg
- before-1.jpg
- after-1.jpg
- before-2.jpg
- after-2.jpg

If before/after images are missing, the website automatically shows elegant placeholders.

IMPORTANT:
For fonts and icons to load, keep internet connected when opening the page.
